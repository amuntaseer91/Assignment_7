    A.2 Practice of Inline Element Tags

    1. bdo:

    BDO stands for Bi-Directional Override. The bdo tag is used to override the current text direction.

    My name is adnan muntaseer

    2. button:

    Inside a button element you can put content, like text or images. This is the difference between this element and buttons created with the input element.

    3. cite:

    The cite tag defines the title of a work (e.g. a book, a song, a movie, a painting, a sculpture, etc.).

    The Scream by Adnan Muntaseer

    4. code:

    It defines a piece of computer code.

    5. kbd:

    It defines keyboard input.

    6. dfn:

    HTML it defines the definiton of the related topics.

    7. object:

    The object tag defines a container for an external resource.

    8. select:

    The select tag is used to create a drop-down list.

    9. span:

    The span tag is an inline container used to mark up a part of a text, or a part of a document.

    My mother has blue eyes.

    10. textarea:

    The textarea tag defines a multi-line text input control.

    11. time:

    We have class at 10.00 am.

    Other Inline elements are:

    acronym, big, em, kbd, map, output, q, samp, script, small, sub, sup, tt, var.
